Pill-shaped editorial button — use for any primary or secondary action in BlueJay Malgudi surfaces.

```jsx
<Button variant="primary" size="md">Submit Feedback</Button>
<Button variant="secondary">View All</Button>
<Button variant="green" iconLeft={<span>＋</span>}>Book Service</Button>
<Button variant="ghost" size="sm">Cancel</Button>
<Button variant="outline">Learn more</Button>
```

Variants: `primary` (maroon gradient), `secondary` (white + border), `green` (garden gradient, for positive/booking actions), `ghost`, `outline`. Sizes `sm | md | lg`. Pass `fullWidth` for form submits, `disabled` to mute.
