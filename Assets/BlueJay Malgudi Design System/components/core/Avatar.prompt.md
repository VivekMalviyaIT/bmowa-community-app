Resident or staff avatar with serif-initial fallback on a warm tinted disc.

```jsx
<Avatar name="Anjali Rao" size={40} tone="maroon" />
<Avatar src="/photo.jpg" name="Anjali Rao" size={48} />
```

`tone` sets the fallback tint (`maroon | garden | sand`). Always pass `name` for accessible initials/alt text.
