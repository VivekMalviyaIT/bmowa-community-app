The editorial surface container — wrap any grouped content (stats, announcements, forms) in a Card.

```jsx
<Card>Plain white surface, soft shadow.</Card>
<Card hover>Lifts on hover — use for clickable list items.</Card>
<Card tone="maroon">Feature card on the maroon gradient.</Card>
<Card tone="garden">Positive / nature-themed highlight.</Card>
<Card tone="glow">Subtle sand backlight bloom behind content.</Card>
```

Tones: `plain | maroon | garden | sand | glow`. Default radius is 24px (`--radius-xl`). Set `padding` to override the default `--space-8`.
