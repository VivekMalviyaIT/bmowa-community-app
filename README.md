# BMOWA Community App

A Modern Glass (Glassmorphism) community management app built with Next.js 16, Tailwind CSS 4, and Framer Motion.

## Features

- **Spotlight**: Community health dashboard with live status cards
- **Snapshot**: Key metrics overview
- **Initiatives**: Community improvement project tracker
- **Handbook**: Guidelines, contacts, and resources
- **Services**: Book community maintenance services
- **Events**: Upcoming community events
- **Feedback**: Submit feedback and suggestions

## Design

- Deep navy/midnight gradient background
- Frosted glass (glassmorphism) components
- Thin white borders with soft drop shadows
- Professional Inter typography in white/silver
- Framer Motion animations
- Responsive: Sidebar (desktop) + Bottom nav (mobile)
